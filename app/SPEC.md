# 备忘录网页 - 规格文档

## 1. 项目概述
- **项目名称**: Memo Pad
- **类型**: 多页Web应用
- **核心功能**: 备忘录工具，分为用户端（只读）和管理端（可管理）
- **目标用户**: 需要快速记录日常事务的用户

## 2. 页面结构

### 用户端 (user.html)
- 仅显示备忘列表
- 可切换完成/未完成状态筛选
- 无任何编辑功能

### 管理端 (admin.html)
- 完整功能：添加、删除、标记完成
- 包含用户端的所有功能

## 3. 视觉规范

### 布局结构
- 居中容器，最大宽度 600px
- 顶部标题区
- 筛选/操作区域
- 备忘列表区域

### 视觉风格
- **配色方案**:
  - 背景: #f5f5f5 (浅灰)
  - 卡片背景: #ffffff
  - 主色调: #4a90d9 (蓝色)
  - 强调色: #e74c3c (删除红)
  - 成功色: #2ecc71 (完成绿)
  - 文字: #333333
  - 次要文字: #888888
- **字体**: "PingFang SC", "Microsoft YaHei", sans-serif
- **圆角**: 8px
- **阴影**: 0 2px 8px rgba(0,0,0,0.1)

### 交互效果
- 按钮悬停: 背景色加深
- 列表项悬停: 轻微上浮 + 阴影增强
- 删除按钮: 红色高亮

## 4. 功能规范

### 用户端功能
1. **查看备忘**: 以列表形式展示所有备忘
2. **状态筛选**: 全部 / 未完成 / 已完成
3. **完成状态**: 显示已完成项的删除线样式

### 管理端功能
1. **添加备忘**: 输入文本后点击添加或按 Enter 键
2. **显示列表**: 所有备忘以列表形式展示
3. **完成标记**: 点击圆圈切换完成状态
4. **删除备忘**: 点击删除按钮移除单条备忘
5. **本地存储**: 使用 localStorage 持久化数据
6. **状态筛选**: 全部 / 未完成 / 已完成

### 数据结构
```javascript
{
  id: timestamp,
  text: "备忘内容",
  completed: false,
  createdAt: Date
}
```

## 5. 技术方案
- 纯 HTML + CSS + JavaScript
- 无外部依赖
- 共享 localStorage 数据
